## 7. DatabaseHelper (lib/database/database_helper.dart)
- Singleton pattern (private constructor + static instance + static getter `instance`)
- _database variable (Database? type)
- _initDatabase() opens/creates student_management.db in documents directory
- onCreate creates students table with 7 columns
- Methods:
  insertStudent (Future<int>)
  getStudentByRegNumber (Future<Student?>)
  getAllStudents (Future<List<Student>>)
  updateStudent (Future<int>)
  deleteStudent (Future<int>)
  getStudentCount (Future<int>)

Recent fixes applied to DatabaseHelper:
- Added `static DatabaseHelper get instance => _instance;` to fix compilation error `DatabaseHelper.instance`
- Fixed `getStudentCount()` rawQuery to use `SELECT COUNT(*) as count FROM students` for reliable `int` casting

SQL Table:
CREATE TABLE students (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  registrationNumber TEXT UNIQUE NOT NULL,
  course TEXT NOT NULL,
  year INTEGER NOT NULL,
  phoneNumber TEXT NOT NULL,
  password TEXT NOT NULL
)
